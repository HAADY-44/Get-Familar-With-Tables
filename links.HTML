      Definition List

      HTML
      HTML (HyperText Markup Language) is the standard language used to create and design web pages. It structures content on the web using elements like headings, paragraphs, links, and images. HTML uses tags to define different parts of a webpage, making it viewable in web browsers.
      CSS
      CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance and layout of HTML elements on a webpage. It allows designers to apply styles like colors, fonts, spacing, and positioning. CSS enhances the visual presentation of web pages, making them more attractive and user-friendly.
